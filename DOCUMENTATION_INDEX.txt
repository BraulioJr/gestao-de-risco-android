                    ��� DICAS PROFISSIONAIS
================================================================================

INSTALAÇÃO:
1. Use Genymotion - É mais rápido que Android Emulator padrão
2. Configure rede em Bridge Mode para acesso real de IP
3. Sempre use "adb install -r" (flag -r = replace existing)
4. Se falhar, execute "adb uninstall" primeiro

DEBUGGING:
1. Execute "adb logcat | grep gestaoderisco" em outro terminal
2. Use "adb shell pm clear" se o app tiver dados corrompidos
3. Limpe Gradle cache se tiver problemas estranhos: "./gradlew clean"

DESENVOLVIMENTO:
1. Firebase Setup é crítico - faça logo no início
2. Implementar LoginActivity antes de outras features
3. Sempre testar offline-first functionality
4. Use WorkManager apenas para tasks em background (não UI)
5. Valide isolamento multi-tenant em TODAS as queries Room
